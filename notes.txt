Notes:
 * MySql like most RDBMS have their own username and passwords. Remember this
   password. You'll need it later.
 * Sometimes there is a seperate script you need to run to secure your install.
   This script will ask you to create a root DB password.
 * The php5-mysql package documentation states that it includes the pdo_mysql
   module. This is the one we need.

Set-up mod_rewrite:

   $ sudo a2enmod rewrite
   $ cd /etc/apache2/sites-available
   $ sudo vi default

Inside "<Directory /var/www/>", find:

      AllowOverride None

Change to:

      AllowOverride All

Save and exit.

   $ sudo service apache2 restart

   $ cd /var/www
   $ ls index.html

Verify the landing page is visible from a Web browser: http://localhost
